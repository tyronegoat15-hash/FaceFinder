# FaceFinder v2

OSINT face search app. Take a photo → find social media + criminal records + web presence.

## Backend Deployment (Railway - Free)

1. Go to https://railway.app → GitHub login
2. New Project → Deploy from GitHub repo
3. Select `face-finder-v2/backend`
4. Set start command: `npm start`
5. Railway gives you a URL like `facefinder.up.railway.app`

## iOS Build

GitHub Actions auto-builds on push. Download .ipa from Actions tab → sideload with Sideloadly.
